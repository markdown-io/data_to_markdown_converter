#Cómo usar el Convertidor

Convierte tablas, documentos, PDF e imágenes a Markdown limpio en segundos. Todo ocurre en tu navegador: sin cuenta, sin instalar nada.

## ¿Qué es Markdown y por qué usarlo con IA?

Markdown es una forma sencilla de dar formato a texto usando símbolos simples (como # para títulos o | para tablas). Cuando pegas una tabla desordenada en una IA como ChatGPT o Claude, muchas veces se descoloca y el modelo la entiende mal. En Markdown la estructura queda clara, la IA la interpreta mejor y además consume menos tokens (las unidades que gastan los modelos).

## Tres formas de convertir

1

Pegar texto o tablas

Copia una tabla de Excel, Google Sheets o Word, o cualquier texto, y pégalo en el cuadro. Presiona Enter y recibes el Markdown al instante. Es la forma más precisa para tablas.

2

Subir un archivo

Usa el botón 📎 (o arrastra el archivo al cuadro) para convertir documentos completos.

.docx.pdf.xlsx .csv.txt
3

Convertir una imagen

Sube una foto o captura (o pégala con Ctrl+V). La herramienta lee el texto de la imagen y lo convierte. Si es una tabla, intenta reconstruirla; si no tiene texto, describe lo que hay en ella.

## Después de convertir

  • Copiar: con un clic copias el Markdown al portapapeles, listo para pegar en tu IA.
  • Descargar: guarda el resultado como archivo .md.
  • Ahorro de tokens: cada conversión te muestra cuántos tokens aproximados ahorras.
~120 tokens → ~45 tokens
Entre más nítida y mejor recortada esté una captura, más preciso es el resultado. Para tablas, siempre que puedas, pegar directamente o subir el archivo original da el resultado más exacto.

## Modo claro y oscuro

Usa el botón 🌙 / ☀️ en la parte superior para cambiar entre tema claro y oscuro. Tu preferencia queda guardada en tu navegador.
